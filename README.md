# Secure Data Foundation – Examination Analytics (Week 2)

This repository documents the secure cloud data foundation created to support the Examination Analytics Blueprint (Week 1).

## What’s Included
- S3 buckets: raw, processed, curated, logs
- Default encryption + versioning + block public access
- Server access logging to centralized logs bucket
- HTTPS-only bucket policy enforcement
- IAM roles following least-privilege access

## Cloud
AWS (S3, IAM), Region: us-east-1
